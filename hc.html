Level:3--Code Breaker

Instructions

How To Play

  1. In this Code Breaker game, the player needs to guess the secret 3-digit code generated by the hacker.
  2. The player enters a 3-digit number in the input field and clicks the "Guess" button.
  3. The game compares the player's guess with the secret code and provides feedback in the format of "X X X"
  4. If any of the digit is matched instead of "X" the particular number will be placed and you have to guess the other digits by placing this digit also
  5. Example: You Have guessed: 123 and the ten's digit is 2 in the code the hacker proivded so it will give "X 2 X" . Now you can try "324" or anything similar to it.
  6. You are given a total of 10 chances to crack the code and everytime you need to give the 3 digit code.
